godirsearch
-----------
```
GoDirSearch -v ~0.0.1
Usage of GoDirSearch:
  -host string
    	Host to brutforce
  -proxy string
    	Use a proxy to brutforce
  -proxyfile string
    	Use a proxy file
  -thread int
    	Number of thread (default 4)
  -tor
    	Brutforce using Tor
  -version string
    	print version (default "v ~0.0.1")
```
